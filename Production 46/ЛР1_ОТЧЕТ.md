## 4. Разделение на уровни (Clean Architecture)

### 4.1. Модели (Models) - Слой данных

**Расположение:** `apps/{module}/models.py`

Модели описывают структуру таблиц базы данных с использованием SQLAlchemy ORM.

**Пример:** [apps/users/models.py](Production 46/backendITSchool/school_site/apps/users/models.py)

```python
from uuid import uuid4
from sqlalchemy import Column, String, Enum, Date
from sqlalchemy.dialects.postgresql import UUID as PostgresUUID
from sqlalchemy.orm import relationship
from school_site.core.db import Base
from school_site.core.models import TimestampMixin
from school_site.core.enums import UserRole

class User(Base, TimestampMixin):
    __tablename__ = "users"
    
    id = Column(PostgresUUID(as_uuid=True), primary_key=True, default=uuid4)
    first_name = Column(String, nullable=True)
    surname = Column(String, nullable=True)
    patronymic = Column(String, nullable=True)
    username = Column(String, unique=True, nullable=False)
    phone_number = Column(String(20), index=True, nullable=True)
    email = Column(String(255), index=True, nullable=False)
    password_hash = Column(String)
    role = Column(Enum(UserRole))
    birth_date = Column(Date, nullable=False)
    
    # Отношения
    refresh_tokens = relationship("RefreshToken", back_populates="user", cascade="all, delete-orphan")
    student = relationship("Student", back_populates="user", uselist=False)
    teacher = relationship("Teacher", back_populates="user", uselist=False)
```

**Особенности:**
- Использование UUID как первичных ключей для распределенных систем
- Миксины для переиспользования общих полей (TimestampMixin, FileMixin)
- Автоматические временные метки created_at/updated_at
- Declarative стиль SQLAlchemy
- Constraints на уровне БД для целостности данных

### 4.2. Схемы (Schemas) - Валидация и сериализация

**Расположение:** `apps/{module}/schemas.py`

Pydantic-схемы для валидации входных данных и сериализации ответов API.

**Пример структуры:**

```python
from pydantic import BaseModel, EmailStr, Field
from uuid import UUID
from datetime import date
from school_site.core.enums import UserRole

# Схема для создания пользователя
class UserCreate(BaseModel):
    first_name: str | None = None
    surname: str | None = None
    patronymic: str | None = None
    username: str = Field(..., min_length=3, max_length=50)
    email: EmailStr
    password: str = Field(..., min_length=8)
    phone_number: str | None = None
    birth_date: date
    role: UserRole

# Схема ответа
class UserResponse(BaseModel):
    id: UUID
    username: str
    email: str
    first_name: str | None
    surname: str | None
    role: UserRole
    
    class Config:
        from_attributes = True  # Для работы с ORM моделями

# Схема для обновления
class UserUpdate(BaseModel):
    first_name: str | None = None
    surname: str | None = None
    phone_number: str | None = None
```

**Особенности:**
- Автоматическая валидация входных данных
- Типизация с использованием типов Python 3.10+
- from_attributes для конвертации ORM моделей в Pydantic
- EmailStr для валидации email
- Field для дополнительных ограничений

### 4.3. Репозитории (Repositories) - Доступ к данным

**Расположение:** `apps/{module}/repositories/`

Паттерн Repository для абстракции работы с базой данных.

**Пример базового репозитория:**

```python
from sqlalchemy.ext.asyncio import AsyncSession
from sqlalchemy import select, update, delete
from typing import TypeVar, Generic, Type, List
from uuid import UUID

T = TypeVar('T')

class BaseRepository(Generic[T]):
    def __init__(self, model: Type[T], session: AsyncSession):
        self.model = model
        self.session = session
    
    async def get_by_id(self, id: UUID) -> T | None:
        result = await self.session.execute(
            select(self.model).where(self.model.id == id)
        )
        return result.scalar_one_or_none()
    
    async def get_all(self, skip: int = 0, limit: int = 100) -> List[T]:
        result = await self.session.execute(
            select(self.model).offset(skip).limit(limit)
        )
        return result.scalars().all()
    
    async def create(self, **kwargs) -> T:
        instance = self.model(**kwargs)
        self.session.add(instance)
        await self.session.flush()
        await self.session.refresh(instance)
        return instance
    
    async def update(self, id: UUID, **kwargs) -> T | None:
        await self.session.execute(
            update(self.model).where(self.model.id == id).values(**kwargs)
        )
        return await self.get_by_id(id)
    
    async def delete(self, id: UUID) -> bool:
        result = await self.session.execute(
            delete(self.model).where(self.model.id == id)
        )
        return result.rowcount > 0
```

**Особенности:**
- Generics для типизации
- Асинхронные операции с БД
- Базовые CRUD операции
- Возможность расширения в наследниках

### 4.4. Сервисы (Services) - Бизнес-логика

**Расположение:** `apps/{module}/services/`

Сервисы содержат бизнес-логику приложения.

**Пример:**

```python
from .repositories.user_repository import UserRepository
from .schemas import UserCreate, UserUpdate
from school_site.core.security import hash_password
from uuid import UUID

class UserService:
    def __init__(self, repository: UserRepository):
        self.repository = repository
    
    async def create_user(self, data: UserCreate):
        # Хешируем пароль
        password_hash = hash_password(data.password)
        
        # Проверяем уникальность email
        existing = await self.repository.get_by_email(data.email)
        if existing:
            raise ValueError("Email already exists")
        
        # Создаем пользователя
        user = await self.repository.create(
            **data.model_dump(exclude={'password'}),
            password_hash=password_hash
        )
        return user
    
    async def get_user(self, user_id: UUID):
        user = await self.repository.get_by_id(user_id)
        if not user:
            raise ValueError("User not found")
        return user
    
    async def update_user(self, user_id: UUID, data: UserUpdate):
        return await self.repository.update(user_id, **data.model_dump(exclude_unset=True))
```

### 4.5. Use Cases - Сценарии использования

**Расположение:** `apps/{module}/use_cases/`

Отдельные use case для сложных операций.

**Пример:**

```python
class RegisterUserUseCase:
    def __init__(self, user_service: UserService, email_service: EmailService):
        self.user_service = user_service
        self.email_service = email_service
    
    async def execute(self, data: UserCreate):
        # Создаем пользователя
        user = await self.user_service.create_user(data)
        
        # Отправляем приветственное письмо
        await self.email_service.send_welcome_email(user.email, user.username)
        
        # Создаем профиль студента/преподавателя в зависимости от роли

            await self.student_service.create_profile(user.id)
        elif data.role == UserRole.TEACHER:
            await self.teacher_service.create_profile(user.id)
        
        return user
```

### 4.6. Маршруты (Routers/Controllers)

**Расположение:** `apps/{module}/router.py`

FastAPI роутеры определяют HTTP endpoints.

**Пример:** [apps/users/router.py](Production 46/backendITSchool/school_site/apps/users/router.py)

```python
from fastapi import APIRouter, Depends, HTTPException, status
from .schemas import UserCreate, UserResponse, UserUpdate
from .services.user_service import UserService
from .depends import get_user_service
from uuid import UUID

router = APIRouter(prefix="/api/users", tags=["users"])

@router.post("/", response_model=UserResponse, status_code=status.HTTP_201_CREATED)
async def create_user(
    data: UserCreate,
    service: UserService = Depends(get_user_service)
):
    """Создание нового пользователя"""
    try:
        user = await service.create_user(data)
        return user
    except ValueError as e:
        raise HTTPException(status_code=400, detail=str(e))

@router.get("/{user_id}", response_model=UserResponse)
async def get_user(
    user_id: UUID,
    service: UserService = Depends(get_user_service)
):
    """Получение пользователя по ID"""
    try:
        user = await service.get_user(user_id)
        return user
    except ValueError:
        raise HTTPException(status_code=404, detail="User not found")

@router.patch("/{user_id}", response_model=UserResponse)
async def update_user(
    user_id: UUID,
    data: UserUpdate,
    service: UserService = Depends(get_user_service)
):
    """Обновление данных пользователя"""
    user = await service.update_user(user_id, data)
    if not user:
        raise HTTPException(status_code=404, detail="User not found")
    return user

@router.delete("/{user_id}", status_code=status.HTTP_204_NO_CONTENT)
async def delete_user(
    user_id: UUID,
    service: UserService = Depends(get_user_service)
):
    """Удаление пользователя"""
    success = await service.delete_user(user_id)
    if not success:
        raise HTTPException(status_code=404, detail="User not found")
```

**Особенности:**
- Dependency Injection через Depends()
- Автоматическая валидация с Pydantic
- Документация Swagger из docstrings
- Явные коды статусов HTTP
- Обработка ошибок через HTTPException

## 5. Зависимости (Dependency Injection)

**Расположение:** `apps/{module}/depends.py` и `core/depends.py`

FastAPI использует DI для внедрения зависимостей.

**Пример:**

```python
from fastapi import Depends
from sqlalchemy.ext.asyncio import AsyncSession
from school_site.core.db import get_db_session
from .repositories.user_repository import UserRepository
from .services.user_service import UserService

def get_user_repository(
    session: AsyncSession = Depends(get_db_session)
) -> UserRepository:
    return UserRepository(session)

def get_user_service(
    repository: UserRepository = Depends(get_user_repository)
) -> UserService:
    return UserService(repository)
```

### 6.1. Подключение к БД

**Файл:** [core/db.py](Production 46/backendITSchool/school_site/core/db.py)

```python
from sqlalchemy.ext.asyncio import create_async_engine, AsyncSession, async_sessionmaker
from sqlalchemy.orm import declarative_base
from school_site.settings import settings

# Создание async engine
engine = create_async_engine(
    settings.database_url,
    echo=settings.debug,
    pool_pre_ping=True,
    pool_size=10,
    max_overflow=20
)

# Session factory
AsyncSessionLocal = async_sessionmaker(
    engine,
    class_=AsyncSession,
    expire_on_commit=False
)

# Base class для моделей
Base = declarative_base()

# Dependency для получения сессии
async def get_db_session() -> AsyncSession:
    async with AsyncSessionLocal() as session:
        try:
            yield session
            await session.commit()
        except Exception:
            await session.rollback()
            raise
        finally:
            await session.close()
```

### 6.2. Базовые миксины

**Файл:** [core/models.py](Production 46/backendITSchool/school_site/core/models.py)

```python
from sqlalchemy import Column, DateTime, String
from sqlalchemy.sql import func

class TimestampMixin:
    """Миксин для автоматических временных меток"""
    created_at = Column(DateTime(timezone=True), server_default=func.now(), nullable=False)
    updated_at = Column(DateTime(timezone=True), server_default=func.now(), onupdate=func.now(), nullable=False)

class CreationTimeMixin:
    """Миксин только для created_at"""
    created_at = Column(DateTime(timezone=True), server_default=func.now(), nullable=False)

class FileMixin:
    """Миксин для файлов"""
    file_name = Column(String, nullable=False)
    file_path = Column(String, nullable=False)
```

### 6.4. Инициализация приложения

**Файл:** [bootstrap.py](Production 46/backendITSchool/school_site/bootstrap.py)

```python
from contextlib import asynccontextmanager
from fastapi import FastAPI
from .core.loggers import set_logging
from .middleware import apply_middleware
from .exceptions import apply_exceptions_handlers
from .router import apply_routes

@asynccontextmanager
async def lifespan(app: FastAPI):
    """Lifecycle события приложения"""
    # Startup
    set_logging()
    scheduler.start()
    
    yield
    
    # Shutdown
    scheduler.shutdown()

def create_app() -> FastAPI:
    app = FastAPI(
        lifespan=lifespan,
        docs_url='/docs',
        openapi_url='/docs.json',
        redirect_slashes=False
    )
    
    app = apply_routes(apply_exceptions_handlers(apply_middleware(app)))
    
    return app
```

```python
from fastapi import FastAPI, Request
from fastapi.responses import JSONResponse
from sqlalchemy.exc import IntegrityError

def apply_exceptions_handlers(app: FastAPI) -> FastAPI:
    @app.exception_handler(IntegrityError)
    async def integrity_error_handler(request: Request, exc: IntegrityError):
        return JSONResponse(
            status_code=400,
            content={"detail": "Database integrity error"}
        )
    
    return app
```
